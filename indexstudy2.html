Case Study #2 Redesign of the USDA (United States Department of Agriculture) web site.

The Problem: The existing USDA Website is cluttered and convoluted with no discernable hierarchy. This creates an unfulfilling user experience. Users become frustrated with the difficulty in finding what they need.

User Persona

The user Presona was created using information gathered from Five user interviews. From this information I was able togather enough overlaping information to create William DeCoster

Site map

This is the preliminary site map that was created from an affinity diagram, Empathy map and heuristic evaluation of the existing Government web site.Final Site Map

Style Tyle

The Style Tyle was created using user insight as well as other referances I then created an inVision moode board cmpiling as much information as possable. I also used the existing government web sit to maintain the important links but removed or combined pages in a more logical order as to simplify the user experience.

The Solution: I will simplify the site by removing duplicated links make link names more descriptive create a organized hierarchy through content chunking and color and imagery.

What I Learned: 1. Get the facts: A through investigation and heuristic evaluation of the existing site gave me a clear understanding of what needed to be corrected and what was correct.

What I Learned: 2. The User Rules: The User Rules: By doing user interviews I was able to get quantified data to determine my user Persona.

What I Learned: 3. Color Is Key: By creating a Style Tyle the creation of a consistent look was acheaved in color, text style, button configeration and imagery.

What I Learned: 4. Walk The Walk: The User Flow was created after a story board was created and User Journey was maped out.

Summary The USDA site like all government sites is created by the guy that got the job with the lowest bid. In general there is too much information that is designed in a manner that does not help the user. This exercise in redesigning a government web site was changeling I was able to use all the tools that i learned in the bootcamp to create a logical redesign.
